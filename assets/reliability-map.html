The Reliability Map
What a reliable agent must deliver, from build to run to review.
Consistency & Determinism
Same task, same decision. Runs reproducible enough to debug and to trust.
Robustness
Behavior holds under perturbed input, adversarial prompts, and shifting tools.
Fault Tolerance & Recovery
Degrade safely, contain the blast radius, return to a known-good state.
Runtime Monitoring
Live signals surface risky behavior early enough to step in.
Failure Attribution
Locate the step, agent, and root cause behind a bad outcome.
Decision Records & Accountability
A durable, inspectable record of each consequential action.
Security Review
Injection, tool and data poisoning, privilege misuse, unsafe actions.
Evaluation & Benchmarks
Comparable, testable reliability metrics, not anecdotes.
